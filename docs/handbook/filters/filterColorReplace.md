# FilterColorReplace

## 调整滤镜
滤镜地址:
> https://s.vipkidstatic.com/vf/plugin/FilterColorReplace/0.0.1.js

滤镜名
> ColorReplaceFilter 

## 属性

| 属性名 | 属性类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| epsilon | 0-1 | 0.1 | 颜色容差 |
| newColor | 0x000000-0xffffff | 0 | 新颜色 |
| originColor | 0x000000-0xffffff | 0 | 原颜色 |
