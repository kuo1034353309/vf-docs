# FilterZoomBlur

## 调整滤镜
滤镜地址:
> https://s.vipkidstatic.com/vf/plugin/FilterZoomBlur/0.0.1.js

滤镜名
> ZoomBlurFilter 

## 属性

| 属性名 | 属性类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| center | vf.Point array | [0,0] | 模糊中心点 |
| innerRadius | number | 0 | 内部模糊 |
| radius | number | -1 | 模糊范围 |
| strength | number | 2 | 模糊强度 |
