# FilterTiltShift

## 调整滤镜
滤镜地址:
> https://s.vipkidstatic.com/vf/plugin/FilterTiltShift/0.0.1.js

滤镜名
> TiltShiftFilter 

## 属性

| 属性名 | 属性类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| blur | number | 0 | 模糊度 |
| end | vf.Point | [0,0] | 模糊结束点 |
| start | vf.Point | [0,0] | 模糊起始点 |
| gradientBlur | number | 0 | 模糊渐变 |
