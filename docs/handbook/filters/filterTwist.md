# FilterTwist

## 调整滤镜
滤镜地址:
> https://s.vipkidstatic.com/vf/plugin/FilterTwist/0.0.1.js

滤镜名
> TwistFilter 

## 属性

| 属性名 | 属性类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| angle | number | 0 | 角度 |
| offset | vf.Point array | [0,0] | 此点描述扭曲的偏移量。 |
| radius | number | 0 | 扭曲范围 |
