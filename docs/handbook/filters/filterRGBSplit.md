# FilterRGBSplit

## 调整滤镜
滤镜地址:
> https://s.vipkidstatic.com/vf/plugin/FilterRGBSplit/0.0.1.js

滤镜名
> RGBSplitFilter 

## 属性

| 属性名 | 属性类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| blue | vf.Point array | [0,0] | 蓝色通道偏移 |
| green | vf.Point array | [0,0] | 绿色通道偏移 |
| red | vf.Point array | [0,0] | 红色通道偏移 |
