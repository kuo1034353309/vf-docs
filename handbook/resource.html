    # 资源

    • 图片、字体、音频、视频、纹理描述、SVG、脚本等资源文件，每个资源对应一个url。例如：jpg的图片，mp3音频文件。
    "assets": {
    "0":{
        "id": 0,
        "name": "bg-homework_frame",
        "type": "image",
        "url": "assets/select1/bg-homework_frame.png"
    },
}
    • 资源在数据中的assets节点下，assets是一个map，key是id， 值是一个object，类型如下
    interface IAsset {
    id?: string;
    name?: string;
    type: AssetType;
    url: string;
}
    • type 支持以下几种类型
    type 类型
    image 图片
    sheet 雪碧图
    svg svg图
    font 字体
    audio 音频
    video 视频
    js 扩展内容
